# HotelRoomManager

HotelRoomManager to aplikacja do zarządzania hotelem, pokojami i gośćmi. Umożliwia dodawanie i usuwanie pokoi, przydzielanie gości, obliczanie przychodu oraz zapis i odczyt danych z plików XML i bazy danych. Projekt wykorzystuje programowanie obiektowe w C#, obsługując różne typy pokoi oraz walidację danych.
